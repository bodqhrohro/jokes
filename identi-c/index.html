Identi-C

About

This a specific software development related joke. See Showtime Komputeishon for the context.

Identi-C™ ([aɪˈdentɪ si] or [aɪˈdentɪk]) is a graphical development environment. Coding primitives are represented as visual blocks which are connected with distinctive lines. They easily and rapidly can be placed on the canvas with the help of useful keyboard macros. Internal schema representation is transparently converted to (a set of) C source code files, which are then compiled with the system-provided C compiler. The development cycle is tremendously simplified. One mouse click starts the program translation and compilation, with the subsequent immediate program execution.

Staggering compatibility technology And-Then-C-Libraries-Can-Be-Used™ ensures any library with a C-compatible interface can be used with a trivial porting effort. The presented prototype includes out-of-the-box GTK+3 and GLib support, which already covers almost all features a desktop application will ever need!

Moreover, Identi-C™ is implemented in itself!
